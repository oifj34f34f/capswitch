# caps_switcher

Switch keyboard layouts using the <kbd>Caps Lock</kbd>, and use the standard Caps Lock function by pressing <kbd>Shift</kbd>+<kbd>Caps Lock</kbd>.

Written by AI.
Supports Windows only.

To run at startup: press <kbd>Win</kbd>+<kbd>R</kbd>, type `shell:startup`, and copy the binary or its shortcut to that folder.
